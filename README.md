## Nota sobre los scripts de Python
Aquí se presentan 3 archivos secuenciales que juntos conforman el proceso completo, desde la importación de datos del archivo xlsx, hasta la exportación de los datos procesados a el SQL Server.

[LimpiezaEq6.ipynb](LimpiezaEq6.ipynb) recibe como entrada el archivo [TEC concentrado_base de datos.xlsx](TEC concentrado_base de datos.xlsx). 

[ProcesamientoFinal.ipynb]([ProcesamientoFinal.ipynb]) recibe como entrada la salida de [LimpiezaEq6.ipynb](LimpiezaEq6.ipynb) y los archivos en [Simulados.zip](Simulados.zip).

[Exportar_a_Nube.py](Exportar_a_Nube.py) recibe como entrada las tablas creadas en [ProcesamientoFinal.ipynb]([ProcesamientoFinal.ipynb]).
